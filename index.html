Git Flow

  1. Create a branch main/master (feature branch)
  2. Make changes to the code
  3. commit the chnages
  4. push the changes to the remote repository on github/ remote feature branch.
  5. create a pull requeston gitgub/ remote feature.
  6. review the pull request
  7. merge the pull request
  8. pull the changes from the remote repository on github/ remote master/main
